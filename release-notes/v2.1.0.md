# PS Transcribe 2.1.0

## What's new

- **A calmer, three-column layout.** Your library on the left, the transcript in the middle, details on the right. Hide the library with ⌘⇧S for a focused reading view.

- **Chat-style transcripts.** Each speaker gets a bubble. Yours on the right, others on the left. Easier to scan than a wall of unbroken text.

- **Quick rename.** Double-click any recording's name to rename it. Return to save, Escape to cancel.

- **Smarter auto-stop.** PS Transcribe listens for actual speech now, not just silence. Voice memos stop after 6 seconds of quiet. Call recordings wait 2 minutes, so a natural pause in a meeting won't end the session.

- **Automatic Notion sync.** Turn on auto-send in Settings and finished recordings will post themselves to your Notion database the moment you stop recording. Add tags afterward without re-sending.

- **Separate Obsidian folders.** Set different vault folders for meetings and voice memos. If a folder isn't configured, the app stops you before you waste a session.

- **Live recording dashboard.** The sidebar footer shows a status dot, recording timer, input device, and a small waveform while you're recording.

## Requirements

Apple Silicon Mac running macOS 26 or newer.

## Install

Download the DMG below, open it, and drag PS Transcribe to Applications. First launch downloads the transcription model (~600MB, one-time).

Note: the source zip files are only the read me. No need to download those.
