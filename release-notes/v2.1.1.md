# PS Transcribe 2.1.1

## What's new

- **Automatic updates.** PS Transcribe now checks for new versions in the background and offers to install them with a single click. You can also trigger a check manually anytime from Settings → Check for Updates.

## Requirements

Apple Silicon Mac running macOS 26 or newer.

## Install

Download the DMG below, open it, and drag PS Transcribe to Applications. First launch downloads the transcription model (~600MB, one-time).

Note: the source zip files are only the read me. No need to download those.
