Apr 20, 2026

v2.1.0

"Silent Quarter"
Current Recommended

Diarization got meaningfully better, the sidebar got quieter, and three long-standing ScreenCaptureKit flakes are gone.

Features

  • Post-session diarization now uses FluidAudio's updated clusterer; fewer false speakers on calls with overlapping talk.
  • Notion integration now supports rollup properties on the destination database.
  • Preferences gained a Vault → Per-recording path template field, with {{year}} / {{month}} / {{title}} tokens.

Interface

  • Library rows stay paper-colored when unfocused instead of shifting to warm gray — matches the rest of the app's quietness.
  • Speaker rename (⌘ E) now shows a 200ms confirmation flash on all affected bubbles.
  • Inspector column collapses below 900px window width.

Fixes

  • Fixed a ScreenCaptureKit hang when switching audio outputs mid-recording.
  • Fixed frontmatter writing double-escaped quotes in participants: for names with apostrophes.
  • Fixed Sparkle update prompt reappearing after dismissal on the same session.

Mar 02, 2026

v2.0.0

"Quiet Chronicle"
Breaking

Full visual rewrite around the Chronicle paper palette. New dual-stream capture pipeline. The app is quieter now — in every sense.

Features

  • Dual-stream capture: microphone + system audio via ScreenCaptureKit, recorded in parallel.
  • Post-session speaker diarization with FluidAudio clusterer.
  • Inline speaker rename (⌘ E).
  • Notion integration: send finished sessions to a database as new pages.

Interface

  • Chronicle design system: paper backgrounds, hairline rules, three-column window.
  • Chat-bubble transcript view with sage rail for the other speaker.
  • Spectral for titles, Inter for body, JetBrains Mono for meta.

Breaking

  • Transcript JSON schema updated; v1 sessions are auto-migrated once on first launch. Back up your vault if you've customized the format.
  • Removed the experimental Whisper backend; Parakeet-TDT is now the only option.

Jan 14, 2026

v1.4.0

Minor

First release with an Obsidian vault integration. Previously transcripts went to a single flat folder; now they land anywhere in your vault.

Features

  • Obsidian vault integration: configurable folder, path templates, YAML frontmatter.
  • Library view gained date-grouped sections.
  • Added ⌘ K quick switcher.

Interface

  • Recording indicator moved from menu bar into the sidebar for visibility.
  • New Preferences window, split into Vault / Audio / Keyboard / About tabs.

Fixes

  • Fixed a permission prompt loop on fresh installs of macOS 14.3.
  • Fixed timestamp drift over hour-long recordings.
