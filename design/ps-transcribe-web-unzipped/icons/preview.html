Build the .icns

On your Mac, from a terminal in the project root:

# 1. Rename to the filenames iconutil expects
mkdir -p AppIcon.iconset
cp icons/iconset-preview/icon_16.png      AppIcon.iconset/icon_16x16.png
cp icons/iconset-preview/icon_16_2x.png   AppIcon.iconset/icon_16x16@2x.png
cp icons/iconset-preview/icon_32.png      AppIcon.iconset/icon_32x32.png
cp icons/iconset-preview/icon_32_2x.png   AppIcon.iconset/icon_32x32@2x.png
cp icons/iconset-preview/icon_128.png     AppIcon.iconset/icon_128x128.png
cp icons/iconset-preview/icon_128_2x.png  AppIcon.iconset/icon_128x128@2x.png
cp icons/iconset-preview/icon_256.png     AppIcon.iconset/icon_256x256.png
cp icons/iconset-preview/icon_256_2x.png  AppIcon.iconset/icon_256x256@2x.png
cp icons/iconset-preview/icon_512.png     AppIcon.iconset/icon_512x512.png
cp icons/iconset-preview/icon_512_2x.png  AppIcon.iconset/icon_512x512@2x.png

# 2. Compile
iconutil -c icns AppIcon.iconset
# → AppIcon.icns
