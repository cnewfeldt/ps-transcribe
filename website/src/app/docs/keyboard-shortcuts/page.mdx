# Keyboard shortcuts

<Lede>PS Transcribe is built for the keyboard. Four shortcuts cover 90% of the app; the rest are for speaker management and navigation.</Lede>

<hr />

## Recording

<ShortcutTable>
  <ShortcutRow
    keys={[{ k: '⌘', tone: 'navy' }, { k: 'R', tone: 'navy' }]}
    label="Start a meeting"
    sub="Captures mic + system audio in parallel."
  />
  <ShortcutRow
    keys={[{ k: '⌘', tone: 'sage' }, { k: '⇧', tone: 'sage' }, { k: 'R', tone: 'sage' }]}
    label="Start a quick memo"
    sub="Mic only. Single-speaker. No diarization pass."
  />
  <ShortcutRow
    keys={[{ k: '⌘' }, { k: '.' }]}
    label="Stop and save"
    sub="Runs VAD, final recognition, and diarization."
  />
  <ShortcutRow
    keys={[{ k: '⌘' }, { k: '⇧' }, { k: '.' }]}
    label="Stop without saving"
    sub="Discards the session. Prompts for confirmation."
  />
  <ShortcutRow
    keys={[{ k: 'Space' }]}
    label="Pause / resume"
    sub="Only available during a recording."
  />
</ShortcutTable>

<Note label="Customize">
  All shortcuts can be rebound in *Preferences → Keyboard*. Your overrides are written to `~/Library/Application Support/PS Transcribe/shortcuts.json`.
</Note>
