**Step 2: Replace the `diagLog` function body.**
Remove the entire FileHandle-based implementation. Replace with:
```swift
func diagLog(_ msg: String) {
    if UserDefaults.standard.bool(forKey: "enableVerboseLogging") {

```

Delete any reference to `/tmp/tome.log`. Remove the `FileHandle` usage, the `seekToEndOfFile`, `write`, `closeFile`, and `createFile` calls from the old diagLog body. There must be zero references to `/tmp/tome.log` or `/tmp/` in this file after the change.

**Step 4: Verify all diagLog call sites still compile.**
Search for `diagLog(` in the file -- every call should still work since the function signature is preserved. Do not change any call site.

**Privacy note:** diagLog messages may contain device names and audio info. Using `.public` privacy is acceptable for debug-level messages that are only visible when the user explicitly enables verbose logging. Paths containing user data should use `.private` -- but existing diagLog calls use string interpolation which will be public. This is acceptable per D-08 since .debug is suppressed by default.
