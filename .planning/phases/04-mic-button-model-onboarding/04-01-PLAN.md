**TranscriptionEngine.swift -- 3 changes:**

1. **Add `activeErrors` computed property** (per D-08, D-09, D-10). Place it right after the `lastError` declaration (~line 22). This aggregates the four error conditions into a list:

```swift
var activeErrors: [String] {
    var errors: [String] = []
    let micStatus = AVCaptureDevice.authorizationStatus(for: .audio)
    if micStatus == .denied || micStatus == .restricted {
        errors.append("Microphone access denied. Enable in System Settings > Privacy > Microphone.")
    }
    if !modelsReady && assetStatus.lowercased().contains("failed") {
        errors.append("Speech model download failed. Check your internet connection.")
    }
    if let err = lastError, !err.isEmpty, !errors.contains(where: { $0 == err }) {
        errors.append(err)
    }
    return errors
}

var hasError: Bool { !activeErrors.isEmpty }
```

Add `import AVFoundation` at the top of the file if not already present (it is likely already there for audio device APIs -- check first).

2. **Fix `prepareModels()` catch block** (~line 72) to reset managers on failure, enabling retry (per RESEARCH.md open question 2). Change the catch block from:

```swift
} catch {
    let msg = "Failed to download models: \(error.localizedDescription)"
    diagLog("[ENGINE-PREP-FAIL] \(msg)")
    lastError = msg
    assetStatus = "Model download failed"
}
```

to:

```swift
} catch {
    let msg = "Failed to download models: \(error.localizedDescription)"
    diagLog("[ENGINE-PREP-FAIL] \(msg)")
    self.asrManager = nil
    self.vadManager = nil
    lastError = msg
    assetStatus = "Model download failed"
}
```

3. **Fix `prepareModels()` guard** (~line 55). The current guard `guard !modelsReady, asrManager == nil else { return }` correctly allows retry when both conditions are met after the catch block resets asrManager. No change needed to the guard itself -- the catch block fix is sufficient.
